# Lenovo_mail
Lavarel实现联想商城

开发语言：php，js，html，css

框架：
lavarel5.3，boostrap3.3.0，jquery



# 主要功能

功能介绍：
实现一般电商网站常用的功能


##后台：

比如商品管理、订单管理、广告管理、轮播图管理、系统配置等等

比如验证码

比如邮箱验证

比如验证


##前台：

浏览物品、添加订单等等
